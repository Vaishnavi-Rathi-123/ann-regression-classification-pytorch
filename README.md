# ANN for Regression & Classification (PyTorch)

This project demonstrates how to build, train, and evaluate an Artificial Neural Network (ANN) using PyTorch for both regression and classification tasks.

---

## 🚀 What this project covers

* Loading and preprocessing dataset
* Converting data into tensors
* Using TensorDataset and DataLoader
* Building ANN architecture
* Training the model using backpropagation
* Tracking training and validation loss
* Saving and loading the best model
* Evaluating model performance

---

## 🧠 Model Architecture

* Input Layer → based on feature size
* Hidden Layer 1 → Linear + ReLU
* Hidden Layer 2 → Linear + ReLU
* Output Layer →

  * Regression → Single neuron
  * Classification → Depends on number of classes
